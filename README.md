# python-port-scanner

## Description
A multi-threaded port scanner built with Python that scans open ports on a target host.

## Features
- Multi-threaded scanning
- Detect open ports
- Saves results to file
- Fast scanning

## Technologies Used
- Python
- Socket
- Threading
- Kali Linux

## Usage
python3 scanner_advanced.py

# Advanced Python Port Scanner

## Overview

This is an advanced multi-threaded port scanner built using Python. The tool scans a target IP address for open ports, performs banner grabbing, and saves the results to a file.

This project was created for cybersecurity learning and ethical hacking practice using Kali Linux.

---

## Features

* Scan target IP
* Scan port range
* Multi-threaded scanning (fast)
* Banner grabbing
* Service detection
* Save results to file
* Simple CLI interface

---

## Requirements

* Python 3
* Kali Linux / Linux / Windows

---

## Installation

Clone the repository:

```
git clone https://github.com/YOUR_USERNAME/port-scanner-advanced.git
cd port-scanner-advanced
```
